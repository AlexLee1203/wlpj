# 05 Release Checklist

## 版本目標

- `v1.0.0`：本地優先的減重計劃日誌 PWA。

## 已完成

- 可新增、編輯、刪除運動紀錄。
- 可依日期查看與管理紀錄。
- 可計算每日總熱量、完成數與完成率。
- 可使用預設運動快速建立紀錄。
- 可匯出與匯入本地 JSON 備份。
- 可清空指定日期的全部紀錄。
- 已具備 PWA manifest 與 service worker。
- 已通過 `npm test` 與 `npm run build`。

## 上線前手動檢查

- [x] 本地 preview 伺服器可正常提供 `index.html`
- [x] `manifest.webmanifest` 可正常存取
- [x] `sw.js` 可正常存取
- [ ] 在桌面瀏覽器操作新增、編輯、刪除、切換日期。
- [ ] 在手機尺寸下檢查表單、預設卡片、備份區排版。
- [ ] 安裝到主畫面後重新開啟 App。
- [ ] 斷網後重新開啟，確認基本畫面仍可載入。
- [ ] 匯出備份後刪除資料，再匯入備份確認資料可恢復。

## 已知限制

- 本地資料仍使用 `localStorage`，容量與可靠性有限。
- 匯入備份目前採覆蓋策略，不做資料合併。
- 熱量數值仍由使用者手動輸入，不做個人化估算。
- 尚未加入登入、同步、統計圖表。

## v2 建議方向

- 改用 `IndexedDB`。
- 增加每週與每月統計。
- 加入依個人條件的熱量估算。
- 支援雲端同步與帳號系統。
